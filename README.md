# Codex: The Agency

**A complete team of AI specialists, built for the OpenAI Codex app.**

Drop a specialist into any Codex session and get deep expertise with personality, process, and real deliverables — not generic prompts.

---

## Install (30 seconds)

```bash
git clone https://github.com/vinceglakas/codex-the-agency
cd codex-the-agency
chmod +x install.sh && ./install.sh
```

That's it. All skills are now available in every Codex session.

## How to Use

### In any Codex thread:
```
$mobile-app-builder Add push notifications to my Expo app with proper permission handling

$growth-hacker Build a referral system with unique codes, tracking, and reward logic

$reddit-community-builder Write a launch post for r/entrepreneur that won't get banned

$app-store-optimizer Write App Store copy for my concert alert app targeting superfans
```

### With AGENTS.md templates:
Drop one of the pre-built `agents-md/` files into your project as `AGENTS.md`. Codex will read it at session start and have the right specialists pre-loaded.

```bash
# For a mobile app project:
cp agents-md/mobile-app.md YourProject/AGENTS.md

# For a SaaS startup:
cp agents-md/saas-startup.md YourProject/AGENTS.md
```
